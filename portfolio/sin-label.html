Sin Label

Sin Label is a social media platform focused on EDM events and the surrounding Fans, Artists and Venues. The web application is very complex and includes many custom features tailored to offer the precise experience and functionality that Sin Label required. The application is Built on the Java Play framework utilizing bootstrap and several other JavaScript plugins.
